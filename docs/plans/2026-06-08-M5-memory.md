# M5 — 0.5.0 Memory Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use `lore:execute` to implement this plan task-by-task.
> **Scope guard:** Do ONLY what is listed here. If you discover adjacent issues, note them as a TODO and continue. Do NOT fix them.

**Goal:** Persist project knowledge across runs — project, decision, command, and architecture memory — and surface it to the planner as context.
**Architecture:** `memory::memory_store` reads/writes JSON files under `.agent/memory/`. The runtime loads memory into the plan context (M3) and updates it at run completion. A `memory` CLI command shows/rebuilds/adds/removes entries.
**Tech Stack:** Rust 2021; reuses M1–M4 deps (serde_json, camino). No new deps.
**Practices:** TDD, typed-interfaces-first, contract-first.
**Required skills:** none.
**Prerequisite:** **M1** (config, errors, run logger), **M2** (ProjectAnalysis feeds project memory), **M3** (planner context consumes memory).
**Design status:** ⚠️ **PROPOSED DESIGN.** SPEC-1 §9 names the memory files (`project.json`, `decisions.json`, `glossary.json`, `commands.json`, `architecture.json`) and §13 names "project/decision/command/architecture memory," but no schemas. The JSON schemas and update triggers below are design decisions to confirm.

**Contracts introduced here (new):** `ProjectMemory`, `DecisionEntry`, `CommandMemory`, `ArchitectureNote`, `MemoryStore`. Files match SPEC-1 §9 names exactly.

```rust
use serde::{Serialize, Deserialize};

#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct ProjectMemory {
    pub name: String, pub language: String, pub package_manager: Option<String>,
    pub last_analyzed: Option<String>, pub source_file_count: usize,
}
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct DecisionEntry {
    pub id: String, pub date: String, pub decision: String,
    pub rationale: String, pub run_id: Option<String>,
}
#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct CommandMemory { pub known_good: Vec<String>, pub known_failing: Vec<String> }
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ArchitectureNote { pub area: String, pub note: String }
#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct GlossaryEntry { pub term: String, pub definition: String }
```
